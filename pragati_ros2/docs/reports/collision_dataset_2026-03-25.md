## How to use this dataset

All events represent pick attempts that the robot **rejected before moving** because the required J5 extension exceeded the collision limit.

**Formula that caused the rejection:**
```
J5_limit = clearance / cos(J3_rad)    [clearance = 0.200 m]
Pick rejected when: J5_cmd > J5_limit
```

**Coordinate conventions:**
- `J3_rot` — tilt joint in rotations (as logged by motor controller)
- `J3_rad` — tilt joint in radians (`J3_rot × 2π`)
- `J3_deg` — tilt joint in degrees (negative = arm tilted downward)
- `J4_m` — lateral sweep joint in metres (positive = right, negative = left)
- `J5_needed_m` — extension depth the planner computed for this cotton boll
- `J5_limit_m` — maximum extension allowed by the collision formula at this J3 angle
- `overshoot_pct` — how much J5_needed exceeded the limit: `(J5_needed / J5_limit - 1) × 100`
