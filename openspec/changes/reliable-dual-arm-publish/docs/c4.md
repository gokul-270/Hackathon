## Level 4: Code-level — `_gz_publish` (triple-publish)

```
_gz_publish(topic, value)
    │
    ├─ [1] subprocess.run(["gz","topic","-t",topic,"-m","gz.msgs.Double","-p","data: {value}"])
    ├─ time.sleep(0.150)
    ├─ [2] subprocess.run(...)
    ├─ time.sleep(0.150)
    └─ [3] subprocess.run(...)
```

## Level 4: Code-level — `executor.execute()` with E-STOP

```
execute(arm_id, applied_joints, ...)
    │
    ├─ [phase 1] publish(j4_topic, j4)   → sleep(0.8s) → check estop
    ├─ [phase 2] publish(j3_topic, j3)   → sleep(0.8s) → check estop
    ├─ [phase 3] publish(j5_topic, j5)   → sleep(1.4s) → check estop
    ├─ [phase 4] publish(j5_topic, 0.0)  → sleep(0.8s) → check estop
    ├─ [phase 5] publish(j3_topic, 0.0)  → sleep(0.8s) → check estop
    └─ [phase 6] publish(j4_topic, 0.0)  → sleep(0.9s) → check estop
              │
              └─ if estop:
                    publish(j3=0, j4=0, j5=0)
                    return {terminal_status: "estop_aborted", ...}
```
